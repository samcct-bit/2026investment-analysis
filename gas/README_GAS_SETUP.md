# 📋 0050 跌破月線自動通知系統 - Google Apps Script (GAS) 部署指南

本指南將引導您如何在 5 分鐘內透過 Google 試算表 (Google Sheets) 與 Google Apps Script (GAS) 完成「0050 跌破月線自動寄發 Email」系統。

---

## 📌 步驟 1：建立 Google 試算表 (Google Sheets)

1. 前往 [Google Sheets](https://sheets.new) 建立一份新的空白試算表。
2. 將試算表命名為：**0050 動態加碼監控**。
3. 在第一列 (Row 1) 填入欄位名稱，在第二列 (Row 2) 精準填入以下公式與資料：

| 儲存格 | 欄位名稱 | 填入內容 / 公式 | 說明 |
| :--- | :--- | :--- | :--- |
| **A1** | 股票代號 | `TPE:0050` | 填寫台股 0050 代號 |
| **A2** | 報價代碼 | `TPE:0050` | GOOGLEFINANCE 讀取目標 |
| **B1** | 目前股價 | - | 標題 |
| **B2** | （公式） | `=GOOGLEFINANCE(A2, "price")` | 自動抓取最新即時/收盤股價 |
| **C1** | 20日均線(月線) | - | 標題 |
| **C2** | （公式） | `=AVERAGE(INDEX(GOOGLEFINANCE(A2, "price", TODAY()-35, TODAY()), 0, 2))` | 抓過去 35 日資料精準計算 20 個交易日均價 |
| **D1** | 狀態判斷 | - | 標題 |
| **D2** | （公式） | `=IF(ISNUMBER(B2), IF(B2<C2, "跌破月線", "觀望"), "讀取中")` | 股價與 20MA 比對判斷 |
| **E1** | 最後發信日期 | - | 標題 |
| **E2** | （留空） | *(留空即可)* | 系統發信後會自動記錄日期 `YYYY-MM-DD` |

---

## 📌 步驟 2：部署 Google Apps Script 程式碼

1. 點擊試算表上方選單：「**擴充功能**」 > 「**Apps Script**」。
2. 將編輯器內原本預設的程式碼清空。
3. 開啟本專案中的 `gas/Code.gs` 檔案，複製全部程式碼，貼入 Apps Script 編輯器中。
4. 點擊上方的「**儲存** (💾 圖示)」或按下 `Ctrl + S`。

---

## 📌 步驟 3：測試發信功能與授權

1. 在 Apps Script 編輯器上方，將執行函式切換為 `testNotifyMail`。
2. 點擊「**執行** (▶️ 鍵)」。
3. 第一次執行時會彈出「**需要權限**」視窗：
   - 點擊「**審查權限**」。
   - 選擇您的 Google 帳號。
   - 點擊「**進階**」 > 點擊「**前往「未命名專案」（不安全）**」。
   - 點擊「**允許**」。
4. 檢查您的 Gmail 信箱，確認收到測試信件。

---

## 📌 步驟 4：設定每日自動定時執行 (Time-Driven Trigger)

1. 點擊 Apps Script 畫面左側選單的「**時鐘圖示 (觸發條件)**」。
2. 點擊右下角「**+ 新增觸發條件**」。
3. 進行以下設定：
   - **選擇要執行的功能**：`check0050AndNotify`
   - **選取活動來源**：`時間驅動`
   - **選取以時間為基準的觸發條件類型**：`日計時器`
   - **選取時間範圍**：`下午 1:00 至 下午 2:00`（因台股 13:30 收盤，此時價格與均線最精準）
4. 點擊「**儲存**」。

🎉 **完成！現在系統每天收盤前/後會自動檢查 0050 是否跌破月線，並在觸發時自動發送信件提醒您分批加碼！**
